## 2) 섹션별 상세 스펙

## [SECTION] LP/MAX/01-header

### A. 목적/메시지
- 목적: 상단 고정 네비게이션과 즉시 데모 신청 CTA 제공
- 기대 행동: `맥스 데모 신청하기` 클릭

### B. 콘텐츠 (Mobile Copy)
- 로고: `MAX`
- 버튼: `맥스 데모 신청하기`

### C. 레이아웃
- 상단 오프셋: `12px`
- 좌우 패딩: `16px`
- 헤더 그룹 높이: `56px`
- 내부 패딩: `pl 16 / pr 12 / py 12`
- CTA 버튼: `132x32`

### D. 스타일
- 배경: `rgba(17,17,19,0.7)` + blur `13px`
- 보더: `1px #2f2f2f`
- 라운드: `9999px`
- 로고 높이: `calc(var(--spacing) * 4)`
- CTA 라벨: `13px / regular / center`

### E. 인터랙션/플로우
- 로고 클릭: 맨 위로 이동 (`#`)
- CTA 클릭: 구글 폼 새 탭 오픈

### F. QA 체크포인트
- iOS Safari / Android Chrome 고정 헤더 동작
- 버튼 라벨 수직 중앙 정렬

## [SECTION] LP/MAX/06-heymax

### A. 목적/메시지
- 목적: 자동화 플로우와 적용 업무 소개

### B. 콘텐츠 (Mobile Copy)
- 섹션 타이틀:
  - `맥스는 스케일업이 필요한`
  - `브랜드를 위해 만들어졌습니다.`
- 하위 타이틀:
  - `반복 업무`
  - `이제는 MAX AI와 함께 해결하세요`
- 설명: `멀티채널 운영은 반복 점검과 보고가 곧 비용입니다. MAX가 대화로 업무 기준을 구조화해 워크플로우를 만들고, 설정된 주기대로 자동 수행합니다.`
- 칩 소제목: `MAX AI는 이런일을 잘해요`

### C. 레이아웃
- 상단 마진: `104px`
- 좌우 패딩: `20px`
- 다이어그램 프레임: `320x500` (고정, 중앙 정렬)
- 다이어그램 상단 간격: `40px`
- 다이어그램-하위 타이틀 간격: `40px`
- 하위 타이틀-설명 간격: `8px`
- 설명-칩소제목 간격: `32px`
- 칩소제목-칩그룹 간격: `32px`

### D. 스타일
- 도트 배경 색: `#464646`
- 배경 도트 영역: `top 151px / height 741px`
- 하위 타이틀: `20px / 400 / 32px / -0.5px`
- 설명: `16px / 400 / 22px / #858585`
- 칩: `h 36`, border `#2f2f2f`, bg `#202020`, text `14px`
- 플로우 점선: `heymax-flow-path` 애니메이션 사용

### E. 인터랙션/플로우
- 점선 흐름 애니메이션(무한 루프)

### F. QA 체크포인트
- 360/375/390에서 다이어그램 중앙축 유지
- `판매 급변 감지` 줄바꿈 방지
